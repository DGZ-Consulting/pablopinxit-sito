# Pablo Pinxit | Visual Artist

Portfolio website for Pablo Pinxit, a visual artist based in Italy.
Showcasing Walls, Shutters, Dadapop, Ikons, and more.

## Tech Stack
- **Framework:** [Astro](https://astro.build)
- **Styling:** [Tailwind CSS](https://tailwindcss.com)
- **Library:** [React](https://reactjs.org) (for interactive galleries/navigation)

## Project Structure
- `/src/pages`: Main site pages (Home, About, Contact, Videos).
- `/src/components`: Reusable UI (Header, Footer, GalleryGrid).
- `/public`: Static assets (Images, Videos).

## Commands
```bash
npm install     # Install dependencies
npm run dev     # Start development server
npm run build   # Build for production
```
